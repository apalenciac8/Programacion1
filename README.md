Este repositorio está dedicado a la recopilación y organización de tareas realizadas durante el curso. Incluye ejercicios, proyectos y prácticas desarrolladas a lo largo del tiempo, con el objetivo de documentar el progreso, facilitar la revisión de contenidos y servir como material de apoyo para el aprendizaje.

Cada carpeta o archivo corresponde a una tarea específica, estructurada de manera clara para su fácil comprensión y acceso.
